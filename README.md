# CarMechanic - Shift Key Mechanics
A full-stack Node.js web application for a car mechanic scheduling system.

## Setup Instructions
1. Clone the repository
2. Run `npm install`
3. Configure your `.env` file based on your local MySQL setup
4. Run `node setup_db.js` to create tables
5. Run `node seed.js` to populate data
6. Start the server with `npm start`
